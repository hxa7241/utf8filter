Description
-----------

A small command-line tool to read stdin and write to stdout only the valid
UTF-8 content. (Invalid meaning: malformed, overlong, surrogate, non-char --
according to RFC-3629 and Unicode 6.1.)

Language: Lua 5.1/5.2 and OCaml 4.00.
